JUYI CHR - Multi-page Next.js site (local scaffold)

How to run locally:
1. npm install
2. npm run dev
3. open http://localhost:3000

How to push to GitHub (from ~/juyi-website):
git add .
git commit -m "Add multi-page site scaffold"
git branch -M main
git push -u origin main
